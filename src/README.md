## 为什么选择 Ditto？

Ditto 是一个面向 Web 的开源操作系统，专为教育、企业演示和低性能设备设计：

- **兼容性优先**：支持 Chrome 80，覆盖老旧设备
- **性能优化**：在树莓派上流畅运行，动画可分级控制
- **安全沙盒**：第三方应用运行在隔离环境中，权限需用户确认
- **易于定制**：主题、壁纸、默认应用均可自定义
