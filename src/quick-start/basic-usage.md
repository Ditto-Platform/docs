# 基本使用

本页面介绍 Ditto WebOS 的基本操作与界面元素。

## 桌面界面

Ditto 启动后呈现一个类似传统操作系统的桌面：

- **桌面区域**：显示图标，双击打开应用
- **任务栏**：底部显示运行中的应用
- **系统托盘**：右下角显示通知中心、控制中心入口
- **开始菜单**：点击左下角按钮打开应用列表

## 核心操作

### 打开应用

方式一：通过开始菜单
1. 点击左下角「开始」按钮
2. 在应用列表中选择应用
3. 点击打开

方式二：通过桌面图标
1. 桌面显示常用应用图标
2. 双击图标打开应用

方式三：通过全局搜索
1. 按 `Ctrl + Space` 打开全局搜索
2. 输入应用名称
3. 点击搜索结果

### 窗口操作

- **移动**：拖拽窗口标题栏
- **调整大小**：拖拽窗口边缘
- **最小化**：点击标题栏最小化按钮
- **最大化**：点击标题栏最大化按钮
- **关闭**：点击标题栏关闭按钮

### 多任务切换

按 `Alt + Tab` 可在运行中的应用之间切换，或点击任务栏图标。

## 系统设置

打开「系统设置」应用可配置：

| 设置项 | 描述 |
|--------|------|
| 主题 | 深色/浅色切换，预设主题选择 |
| 动画档位 | 关闭/微妙/标准/丰富，适应不同性能设备 |
| 窗口布局 | 浮动/平铺/磁吸 |
| 壁纸 | 自定义桌面背景 |
| 应用管理 | 查看/卸载已安装应用 |

## 应用市场

「Ditto Market」是官方应用市场：

1. 浏览分类应用
2. 查看应用详情、评分、评论
3. 安装第三方应用
4. 更新已安装应用

## 通知中心

点击托盘区铃铛图标打开通知中心：

- 查看系统通知
- 清除单个或全部通知

## 控制中心

点击托盘区滑块图标打开控制中心：

- 快速调节亮度、音量
- 切换深色/浅色模式
- 快捷电源操作（锁屏、睡眠、重启）

## 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + Tab` | 切换应用 |
| `Ctrl + Space` | 全局搜索 |
| `Ctrl + L` / `Meta + L` | 锁屏 |
