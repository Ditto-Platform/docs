# 快速开始

欢迎使用 Ditto WebOS！本章节将帮助你快速上手。

## 目录

- [安装](/quick-start/installation) - 环境准备与项目启动
- [基本使用](/quick-start/basic-usage) - 系统界面与核心操作

## 系统要求

| 项目 | 要求 |
|------|------|
| Node.js | >= 18.0.0 |
| pnpm | >= 9.0.0 |
| 浏览器 | Chrome 80+ / Firefox 75+ / Safari 13+ |
| 分辨率 | >= 320px 宽度 |

::: tip 低性能设备支持
Ditto 在树莓派、老旧 PC 上也能流畅运行，只需将动画档位设置为「关闭」或「微妙」。
:::
