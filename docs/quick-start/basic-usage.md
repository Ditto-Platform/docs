# 基本使用

本页面介绍 Ditto WebOS 的桌面界面、核心操作、内置应用与全局快捷键，帮助你快速上手 Web 操作系统。

## 系统界面介绍

Ditto 启动后呈现一个类似传统操作系统的桌面环境，主要由以下 5 个区域组成：

### 桌面区域

桌面是 Ditto 的主工作区，提供：

- **桌面图标**：常用应用的快捷入口，双击即可打开
- **壁纸**：可在「系统设置 → 壁纸」中自定义，支持纯色、渐变、图片
- **右键菜单**：右键桌面空白处可快速访问设置、刷新、更改壁纸等

### 任务栏

位于屏幕底部，从左至右依次为：

| 区域 | 元素 | 说明 |
|------|------|------|
| 左侧 | 开始按钮 | 点击打开开始菜单 |
| 中部 | 运行中应用图标 | 显示所有打开的应用，点击切换/最小化/恢复 |
| 右侧 | 系统托盘 | 显示通知中心、控制中心、时钟、网络等 |

::: tip 应用指示
运行中的应用在任务栏会高亮显示，最小化的应用图标会显示下划线指示器。
:::

### 开始菜单

点击左下角「开始」按钮打开，包含：

- **应用列表**：按字母或分类排列的全部已安装应用
- **搜索框**：顶部搜索框可快速查找应用
- **电源菜单**：右下角提供锁屏、睡眠、重启、关机选项
- **用户头像**：显示当前用户信息

### 通知中心

点击托盘区铃铛 🔔 图标打开，提供：

- 查看系统通知（应用消息、系统提示、权限申请等）
- 清除单个通知（点击通知右侧 ✕）或全部清除（点击「清除全部」）
- 通知历史记录
- 勿扰模式开关

### 控制中心

点击托盘区滑块 ⚙ 图标打开，提供快速设置：

| 控件 | 说明 |
|------|------|
| 亮度 | 滑块调节屏幕亮度 |
| 音量 | 滑块调节系统音量 |
| 主题模式 | 深色 / 浅色 / 跟随系统切换 |
| 飞行模式 | 开关网络连接 |
| 蓝牙 | 开关蓝牙（预留） |
| 电源 | 锁屏 / 睡眠 / 重启 / 关机 |

## 核心操作

### 启动应用

Ditto 提供 3 种启动应用的方式。

#### 方式一：通过开始菜单

1. 点击左下角「开始」按钮（或按 `Meta` 键）
2. 在应用列表中浏览查找，或在顶部搜索框输入应用名称
3. 点击应用图标即可启动

#### 方式二：通过桌面图标

1. 桌面显示常用应用图标
2. 双击图标打开应用

::: tip 添加桌面快捷方式
在开始菜单的应用图标上右键，选择「添加到桌面」即可创建桌面快捷方式。
:::

#### 方式三：通过全局搜索

1. 按 `Ctrl + Space` 打开全局搜索
2. 输入应用名称或关键词
3. 在搜索结果中点击应用启动

全局搜索不仅可搜索应用，还可搜索文件、设置项、网页建议。

### 窗口管理

每个应用打开后会以独立窗口形式呈现，支持以下操作：

| 操作 | 方式 |
|------|------|
| **移动窗口** | 拖拽窗口标题栏到目标位置 |
| **调整大小** | 拖拽窗口四边或四角 |
| **最小化** | 点击标题栏右侧 `—` 按钮，或点击任务栏图标 |
| **最大化 / 还原** | 点击标题栏右侧 `▢` 按钮，或双击标题栏 |
| **关闭** | 点击标题栏右侧 `✕` 按钮 |
| **磁吸布局** | 拖拽窗口到屏幕左/右边缘触发磁吸 |

::: tip 窗口布局模式
通过「系统设置 → 窗口布局」可切换：
- **浮动**：自由摆放，传统桌面体验
- **平铺**：自动平铺，类似 i3 / sway
- **磁吸**：拖拽时自动吸附，类似 Windows Snap
:::

### 多任务切换

Ditto 提供多种多任务切换方式：

- **`Alt + Tab`**：按住 `Alt` 不放，反复按 `Tab` 在运行中的应用间循环切换，松开 `Alt` 后切换到选中应用
- **任务切换器**：同时显示所有窗口的缩略图预览
- **任务栏点击**：直接点击任务栏应用图标切换到该应用
- **窗口拖拽**：将窗口拖拽到屏幕边缘磁吸分屏

::: tip 任务切换器
按住 `Alt` 后按 `Tab` 切换时，屏幕中央会显示所有应用窗口的实时缩略图，方向键或鼠标可选择目标应用。
:::

### 全局搜索

按 `Ctrl + Space` 可随时打开全局搜索（即使在应用全屏时也可触发）：

- **应用搜索**：输入应用名称快速启动
- **文件搜索**：搜索虚拟文件系统中的文件
- **设置项搜索**：跳转到对应的系统设置页面
- **网页建议**：将搜索词作为网页查询建议（可选）

::: tip 全局搜索定制
全局搜索结果来源可在「系统设置 → 搜索」中配置，可关闭特定来源。
:::

### 锁屏

按 `Meta + L`（或 `Ctrl + L`）可立即锁屏：

- 锁屏后显示时钟与解锁按钮
- 输入密码或 PIN 后解锁
- 锁屏期间后台应用继续运行
- 可在「控制中心」点击锁屏图标手动触发

::: warning 自动锁屏
可在「系统设置 → 安全」中配置自动锁屏时间（1 / 5 / 15 / 30 分钟）。
:::

### 主题切换

Ditto 提供多种主题切换方式：

#### 方式一：控制中心快速切换

点击托盘区控制中心图标，点击「深色 / 浅色 / 跟随系统」按钮快速切换。

#### 方式二：系统设置

打开「系统设置 → 主题」，可选择：

| 选项 | 说明 |
|------|------|
| 浅色模式 | `dittoLight` 预设，明亮配色 |
| 深色模式 | `dittoDark` 预设，护眼配色 |
| 跟随系统 | 根据浏览器 `prefers-color-scheme` 自动切换 |
| 自定义主题 | 加载第三方主题（如 `com.ditto.theme.nord`、`com.ditto.theme.forest`） |

#### 主题引擎特性

- **CSS 变量驱动**：通过 `--ditto-*` 自定义属性，运行时切换无闪烁
- **组件级 override**：每个组件可独立覆盖主题 token
- **动画档位**：可选 none / subtle / normal / expressive，适应不同性能设备

::: tip 教育场景
低性能设备推荐设置动画档位为「关闭」或「微妙」，可显著提升响应速度。
:::

## 内置应用一览

Ditto 默认提供 9 个系统应用，覆盖日常使用需求：

| # | 应用 | 图标 | 说明 |
|---|------|------|------|
| 1 | **文件管理器** (File Manager) | 📁 | 浏览虚拟文件系统、管理文件与目录、上传/下载、跨应用分享 |
| 2 | **系统设置** (Settings) | ⚙ | 主题、壁纸、动画档位、窗口布局、应用管理、安全设置 |
| 3 | **关于** (About) | ℹ | 系统版本、内核信息、Cell 状态、许可证、技术栈 |
| 4 | **市场** (Ditto Market) | 🛍 | 官方应用市场，浏览、搜索、安装、更新第三方应用 |
| 5 | **终端** (Terminal) | ⌨ | 命令行工具，支持文件操作、应用启动、系统命令 |
| 6 | **文本编辑器** (Text Editor) | 📝 | 轻量级代码/文本编辑器，支持语法高亮、多标签 |
| 7 | **系统监视器** (System Monitor) | 📊 | 实时监控 CPU、内存、网络、Cell 资源占用 |
| 8 | **聊天** (Chat) | 💬 | 内置即时通讯演示应用，基于 Cell 后端实时通信 |
| 9 | **资源中心** (Resource Hub) | 📚 | 系统资源、文档、教程、开发者资源一站入口 |

::: tip 应用管理
所有内置应用均可在「系统设置 → 应用管理」中查看详情、卸载（部分核心应用不可卸载）、设置默认关联。
:::

### 内置应用详解

#### 📁 文件管理器

文件管理器提供虚拟文件系统（VFS）的图形化操作：

- 浏览 `/home`、`/documents`、`/downloads`、`/apps` 等目录
- 支持「列表视图」与「图标视图」两种显示模式
- 文件操作：复制、移动、重命名、删除、属性查看
- 上传本地文件到 VFS，下载 VFS 文件到本地
- 跨应用分享：右键 →「打开方式」选择其他应用

#### ⚙ 系统设置

系统设置是配置 Ditto 行为的中枢，包含以下分类：

| 设置项 | 描述 |
|--------|------|
| 主题 | 深色/浅色切换，预设主题选择 |
| 动画档位 | 关闭 / 微妙 / 标准 / 丰富，适应不同性能设备 |
| 窗口布局 | 浮动 / 平铺 / 磁吸 |
| 壁纸 | 自定义桌面背景 |
| 应用管理 | 查看/卸载已安装应用 |
| 默认应用 | 文件类型关联、URL 协议关联 |
| 安全 | 锁屏时间、权限策略、沙盒模式 |
| 网络 | API 端点、代理、超时配置 |
| 通知 | 通知权限、勿扰模式 |
| 关于 | 系统信息 |

#### 🛍 市场（Ditto Market）

「Ditto Market」是官方应用市场：

1. 浏览分类应用（工具、游戏、主题、开发等）
2. 查看应用详情、截图、评分、评论
3. 一键安装第三方 `.dit` 应用包
4. 检查并更新已安装应用
5. 卸载已安装应用

::: warning 第三方应用安全
市场中的应用均经过签名校验，但仍建议查看应用申请的权限（`fs:read`、`net:fetch` 等），首次启动时会通过权限对话框确认。
:::

#### ⌨ 终端

终端提供命令行操作能力：

- 文件操作：`ls`、`cd`、`cat`、`mkdir`、`rm`
- 应用启动：`open <app-id>` 启动指定应用
- 系统命令：`ps`、`kill`、`info`、`theme`
- 网络命令：`curl`、`ws`
- 支持 Tab 自动补全、上下方向键浏览历史

#### 📝 文本编辑器

轻量级代码/文本编辑器：

- 多标签页编辑
- 语法高亮（基于 Prism.js）
- 自动缩进、行号显示
- 文件保存到 VFS
- 支持加载 VFS 中的任意文本文件

#### 📊 系统监视器

实时监控 Ditto 系统状态：

- **CPU 占用**：主线程任务负载
- **内存占用**：JS Heap / DOM 节点 / 事件监听器
- **网络流量**：API 请求 / WebSocket 消息速率
- **Cell 状态**：每个运行中 Cell 的资源占用、状态（loading / active / paused / stopped）
- **应用列表**：所有运行中应用的资源排行

#### 💬 聊天

内置即时通讯演示应用：

- 基于 Cell 后端实时通信（WebSocket）
- 多用户会话演示
- 消息历史、在线状态、表情支持
- 可作为第三方应用开发的实时通信参考

#### 📚 资源中心

资源中心是系统资源与文档的统一入口：

- 官方文档链接（架构、API、教程）
- 开发者资源（SDK 文档、CLI 命令）
- 示例应用集合
- 社区资源、视频教程
- 常见问题与故障排查

## 快捷键一览表

### 全局快捷键

| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `Alt + Tab` | 切换应用 | 按住 Alt 反复按 Tab 循环 |
| `Ctrl + Space` | 全局搜索 | 任何位置可触发 |
| `Meta + L` | 锁屏 | 等同于 `Ctrl + L` |
| `Ctrl + L` | 锁屏 | 备用快捷键 |
| `Meta` | 打开开始菜单 | 单独按 Meta (Win/Super) 键 |
| `Meta + D` | 显示桌面 | 最小化所有窗口 |
| `Print Screen` | 截图（预留） | 触发截图工具 |

### 窗口快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + F4` | 关闭当前窗口 |
| `Alt + Space` | 打开窗口菜单（最小化/最大化/关闭） |
| `Win + ↑` | 最大化窗口 |
| `Win + ↓` | 最小化 / 还原窗口 |
| `Win + ←` | 窗口磁吸到左半屏 |
| `Win + →` | 窗口磁吸到右半屏 |

### 应用通用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新建 |
| `Ctrl + O` | 打开文件 |
| `Ctrl + S` | 保存 |
| `Ctrl + W` | 关闭当前标签 / 文件 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + F` | 查找 |
| `Ctrl + P` | 打印（预留） |

::: tip 快捷键定制
开发者可在 `apps/shell/src/main.ts` 中通过 `useKeyboard` composable 注册自定义全局快捷键。详见 [开发指南](/development/)。
:::
