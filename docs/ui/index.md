# UI & 主题

本章节介绍 Ditto WebOS 的 UI 组件库与主题定制。

## UI 组件

Ditto 提供一套 Vue 组件库：

| 组件 | 描述 |
|------|------|
| `DWindow` | 窗口容器 |
| `DIcon` | 统一图标渲染 |
| `DDialog` | 对话框 |
| `DNotification` | 通知气泡 |
| `DContextMenu` | 右键菜单 |
| `DTaskbar` | 任务栏 |
| `DDesktop` | 桌面区域 |
| `DStartMenu` | 开始菜单 |
