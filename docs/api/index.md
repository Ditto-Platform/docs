# API 参考

本章节提供 Ditto WebOS 的 API 文档。

## API 概览

Ditto 提供两类 API：

| 类型 | 协议 | 用途 |
|------|------|------|
| 服务端 API | HTTP/WebSocket | 应用管理、Market、用户认证 |
| 前端 API | TypeScript SDK | fs、notification、ipc、clipboard |
